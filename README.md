# dandydomains

## install

1. `npm install`
2. add mashape api key to `.env` (see `.env-sample`) _you'll need to set yourself up with a [mashape account](https://market.mashape.com/domainr/domain)_

## start

1. `npm start`
2. its now running at http://localhost:3000

## dev

`npm run dev`

### using
